# 🌦️ Weather App

A simple weather app built with **HTML**, **CSS**, and **JavaScript** using the **OpenWeatherMap API**.

---

## 📌 Features
- 🔍 Search weather by city name
- 🌡️ Displays temperature, condition, humidity, and wind speed
- 🎨 Weather-themed background
- 📱 Responsive design
